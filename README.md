# Mutual Fund Analytics & Investor Insights

## Bluestock Fintech Capstone Project

### Project Overview

This project is an end-to-end Mutual Fund Analytics platform developed during the Bluestock Fintech Capstone Program. It demonstrates the complete data analytics lifecycle, including ETL, Exploratory Data Analysis (EDA), financial performance analysis, advanced analytics, and interactive dashboard development using Power BI.

## Features

* ETL Pipeline
* Data Cleaning & Validation

* CAGR Calculation

* Maximum Drawdown

## How to Run

### Clone Repository

```bash
git clone <repository-url>
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run ETL

```bash
python data_ingestion.py
```

### Run Analytics

```bash
python recommender.py
```
